Questo é il Capston Projects.
Vi presento Crazy Pinball! Un platform 2D a scorrimento laterale ambientato in un Pinball!
Tu sei Bill la Palla per gli amici solo Bill.
Un giorno tutti i Pinball della sala Arcade Crazy Pinball vengono colpiti da un fulmine mandandoli in tilt, e sprigionando un'entitá
malvaggia di nome Tilt Zapper, ci penserá Bill a viaggiare di Pinball in Pinball per sconfiggere Tilt Zapper

// CONTENUTI ATTUALI //-/-/-/-/-/-/-/-/-/-/-/-/-/
- Player: move, break, jump
- Objects: flipper, bumper
- 1 Basic Enemy: si muove a destra e sinistra daneggiando il player
- Screens: Title, Level Select, Victory, GameOver
- 2 Full Levels
- Saving System, finito un livello sblocca il successivo e rimane salvato

// COMANDI //-/-/-/-/-/-/-/-/-/-/-/-/-/-/-/-/-/-/

PLAYER--------
A -> left
D -> right
W -> break

FLIPPERS------
MOUSE LEFT BUTTON -> left flipper
MOUSE RIGHT BUTTON -> right flipper
